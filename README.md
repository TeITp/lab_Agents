## 3. Робота у віртуальному середовищі (VENV)

**⭐ Що вивела остання команда (`pip show requests`) та чому?**
Команда вивела попередження, що пакет `requests` не знайдено (або показала глобальну версію). Це сталося тому, що команда `deactivate` вимкнула віртуальне середовище `my_env`, і ми повернулися до глобального середовища системи, де ця бібліотека була раніше видалена.

**⭐ Які папки потрібно ігнорувати для VENV середовища:**

## 4. Робота з Pipenv

**⭐ Які команди можна виконувати за допомогою `pipenv`:**
* **pipenv install** — встановлення всіх залежностей або конкретного пакета.
* **pipenv shell** — активація віртуального середовища в поточному терміналі.
* **pipenv run** — запуск скриптів або команд всередині середовища.
* **pipenv graph** — вивід дерева залежностей пакетів.
* **pipenv check** — перевірка проекту на вразливості та відповідність версій.
* **pipenv lock** — генерація файлу Pipfile.lock для фіксації версій.
* **pipenv --venv** — отримання шляху до папки віртуального середовища.
* **pipenv --rm** — повне видалення поточного віртуального середовища.
* **pipenv update** — оновлення пакетів до останніх доступних версій.

**⭐ Що знаходиться у файлах `Pipfile` та `Pipfile.lock`:**
* **Pipfile:** Містить загальні налаштування середовища (версія Python) та списки встановлених пакетів (основні та для розробки).
* **Pipfile.lock:** Містить точні версії всіх встановлених залежностей та їхні хеш-суми для забезпечення абсолютно ідентичного середовища на інших комп'ютерах.

## 5. Робота зі змінними середовища

**⭐ Що буде, якщо виконати скрипт без активації віртуального середовища?**
Виникне помилка `KeyError: 'IT_TEST'`. Без активації середовища `pipenv` файл `.env` не завантажується автоматично в змінні оточення `os.environ`, тому скрипт не зможе знайти цей ключ.

**⭐ Програма, створена за допомогою ШІ та запущена в середовищі Poetry:**
```python
from flask import Flask
from jikanpy import Jikan

app = Flask(__name__)
jikan = Jikan()

@app.route('/')
def index():
    try:
        # Беремо дані з попереднього завдання
        anime = jikan.anime(54595)
        title = anime['data']['title']
        score = anime['data']['score']
        return f"<h1>Аніме: {title}</h1><p>Оцінка: {score}</p>"
    except Exception as e:
        return str(e)

if __name__ == '__main__':
    app.run(debug=True)
```
